# Infra as Go

*Using Go code as the description language for infrastructure provisioning and operations.*

Your infrastructure deserves a real programming language, not a templating engine with anxiety.
Drop the YAML indentation nightmares. Drop the HCL halfway house.
Write infrastructure in Go: typed, tested, and unapologetically compiled.
